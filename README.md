# GameMishen

Игра "Mishen!"
Логика игры проста: 
даётся 5 выстрелов, за каждый из них начисляется от 0 до 5 баллов, в зависимости от сектора, в который ты попал. Набери 10 и больше, если ты не лузер! 

Правила распределения баллов:
Центр - 5 баллов,
2 сектор - 4 балла,
3 сектор - 3 балла,
4 сектор - 2 балла,
5 сектор - 1 балл,
Мимо - 0 баллов.
